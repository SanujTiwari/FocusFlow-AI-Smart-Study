# FocusFlow - AI Smart Study Assistant

FocusFlow is an AI-powered study planner that helps students manage their syllabus, create study schedules, generate flashcards, and track learning progress efficiently.

## Features

### AI Study Planner
- Smart timetable generation based on exams and subject difficulty
- Automatic rescheduling of missed study sessions
- Priority-based study allocation and revision planning

### AI Flashcards & Spaced Repetition
- AI-generated flashcards using Gemini
- SM-2 spaced repetition algorithm
- Interactive flashcard learning experience
- Custom flashcard creation

### Smart Syllabus Import
- Import syllabus from PDF, Images, or Text
- AI-powered chapter extraction
- Topic selection and progress tracking

### Analytics Dashboard
- Study streaks and productivity insights
- Subject-wise study analytics
- Interactive charts and reports
- Built-in Pomodoro timer

## Tech Stack

**Frontend:** React.js, Vite, Tailwind CSS, Zustand, Framer Motion, Recharts

**Backend:** Node.js, Express.js, PostgreSQL, Sequelize

**AI:** Google Gemini API
